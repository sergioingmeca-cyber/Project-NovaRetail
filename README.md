# 🛍️ NovaRetail | Retail Sales Performance Analysis

## 📖 Resumen Ejecutivo

Las empresas del sector retail generan grandes volúmenes de información provenientes de ventas, clientes y productos. Transformar estos datos en información útil es fundamental para identificar oportunidades comerciales, optimizar la estrategia de ventas y respaldar la toma de decisiones.

En este proyecto se desarrolló un análisis exploratorio de datos (EDA) utilizando Python para evaluar el comportamiento de las ventas, identificar tendencias comerciales y descubrir patrones que permitieran comprender mejor el desempeño del negocio.

---

# 🎯 Objetivo del Proyecto

Analizar la información comercial de NovaRetail mediante técnicas de análisis exploratorio de datos para identificar patrones de ventas, comportamiento de clientes y oportunidades de mejora que apoyen la toma de decisiones basada en datos.

---

# 📂 Descripción del Dataset

El conjunto de datos contiene información relacionada con las operaciones comerciales de una empresa del sector retail, incluyendo variables asociadas a clientes, productos, transacciones y ventas.

Durante el proyecto se realizó:

* Exploración inicial del dataset.
* Limpieza y preparación de datos.
* Tratamiento de valores faltantes.
* Análisis estadístico descriptivo.
* Identificación de patrones y tendencias.
* Visualización de indicadores relevantes.
